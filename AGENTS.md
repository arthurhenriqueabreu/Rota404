# RotaLucro - Instruções para agentes

## Objetivo
Criar um MVP para o Hackathon Softville Senac, tema logística no comércio.

## Produto
RotaLucro é um assistente de entregas para pequenos mercados. A interface simula uma conversa de WhatsApp onde o comerciante cadastra mercado, veículo e entregas. No final, o sistema calcula rota mockada, custo real, frete recebido e lucro/prejuízo.

## Diferencial
Google Maps mostra o caminho. O RotaLucro mostra se a entrega compensa financeiramente.

## Prioridade máxima
Entregar uma demo funcional rapidamente.

## Escopo do MVP
- Tela simulada de WhatsApp
- Fluxo guiado com perguntas pré-definidas
- Cadastro de mercado
- Cadastro de veículo
- Cadastro de entregas
- Cálculo mockado
- Resumo final com lucro/prejuízo
- Botão "Carregar demo"

## Não fazer agora
- backend
- banco de dados
- login
- pagamento
- API real do WhatsApp
- Google Maps real
- IA real funcionando
- arquitetura complexa
- dependências desnecessárias

## Regra de trabalho
Faça alterações pequenas e testáveis.
Não reescreva o projeto inteiro sem necessidade.
Se for alterar muito código, explique antes.
